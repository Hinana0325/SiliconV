# core — 平台核心

SiliconV 的心脏。包含 VM 管理器、中断控制器、设备树生成器和 CPU 生命周期管理。

## 模块

| 模块 | 描述 |
|------|------|
| [vm/](vm/) | VM 主循环、启动镜像解析、启动桩 |
| [irq/](irq/) | GICv3 中断控制器模拟 |
| [memory/](memory/) | DTB（设备树）运行时生成器 |
| [object/](object/) | PSCI — CPU 电源状态协调 |
| [dma/](dma/) | DMA 缓冲区管理 |
| [scheduler/](scheduler/) | vCPU 调度器 |

## 架构

```
┌─────────────────────────────────────┐
│           VM Machine (machine.c)     │
│  ┌────────┬────────┬──────────────┐ │
│  │  GICv3 │  PSCI  │  DTB 生成    │ │
│  │ (irq/) │(object/)│ (memory/)   │ │
│  └────────┴────────┴──────────────┘ │
│  ┌────────────────────────────────┐ │
│  │  启动镜像解析器 (bootimg.c)     │ │
│  └────────────────────────────────┘ │
└─────────────────────────────────────┘
```

## 关键设计决策

- **直启模式** — 无需引导加载程序；解析 boot.img 后直接跳转到内核
- **运行时 DTB** — 启动时根据规范生成设备树，而非构建时预置
- **仅 GICv3** — 不支持旧版 GICv2；简化代码并匹配现代内核
- **PSCI 通过 HVC** — 客户内核通过 HVC（Hypervisor Call）调用 PSCI，而非 SMC
