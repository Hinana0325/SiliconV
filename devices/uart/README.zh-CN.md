# devices/uart — PL011 UART

ARM PL011 UART 模拟，用于客户串口控制台输出。这是主要的调试和早期启动输出通道。

## 概述

PL011 是标准 ARM UART。SiliconV 实现了最小寄存器集，支持：

- 字符输出（客户 → 宿主终端）
- 字符输入（宿主键盘 → 客户）
- 基本中断信号

## MMIO 布局

| 地址 | 寄存器 | 描述 |
|------|--------|------|
| `0x10000000` | DR | 数据寄存器（读/写） |
| `0x10000018` | FR | 标志寄存器（TX/RX 状态） |
| `0x10000024` | IBRD | 整数波特率 |
| `0x10000028` | FBRD | 小数波特率 |
| `0x1000002C` | LCR_H | 线路控制 |
| `0x10000030` | CR | 控制寄存器 |
| `0x10000038` | IMSC | 中断掩码 |
| `0x10000044` | ICR | 中断清除 |

IRQ：**32**（第一个 SPI）

## 文件

| 文件 | 描述 |

## 客户连接

默认情况下，UART 连接到宿主的 stdout/stdin。在 CLI 模式下，字符直接流向终端。在 GUI 模式（Cocoa/Qt）下，字符路由到终端组件。
